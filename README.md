> 这个项目是对 [josdirksen/learning-threejs](https://github.com/josdirksen/learning-threejs) 的一次重写。  
> 仅供个人学习用。

>
## Preview
you can click here to preview<a href="https://yufengjie97.github.io/learning-threejs/#/" target="_blank">preview</a>
## Install  
      pnpm install
## Run Dev
      pnpm run dev
